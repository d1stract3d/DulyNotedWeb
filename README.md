# DulyNotedWeb

DulyNotedWeb is a CRUD demo made with Spring Framework using Jdbc and MYSQL.
It is the primitive ancestor of [DulyNoted for iOS](https://github.com/d1stract3d/DulyNoted/tree/master/dulyNoted)
